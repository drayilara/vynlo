# Vynlo — AIGC Platform

## Project Overview
Vynlo is an AI-Generated Content (AIGC) platform built with Next.js.

## Tech Stack
- **Framework**: Next.js (App Router) with Turbopack
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Package Manager**: npm

## Project Structure
- `src/app/` — App Router pages and layouts
- `src/` — Source code root
- `public/` — Static assets

## Development
- Dev server: `npm run dev` (runs on port 3000 by default)
- Build: `npm run build`
- Lint: `npm run lint`

## Conventions
- Use the `@/*` import alias for imports from `src/`
- Follow Next.js App Router conventions for routing and layouts
- Use Tailwind CSS for styling — avoid CSS modules or styled-components
